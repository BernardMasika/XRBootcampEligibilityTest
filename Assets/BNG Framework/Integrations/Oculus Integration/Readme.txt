Installation Instructions : 

1. Download and import the Oculus Integration Kit into your project : https://assetstore.unity.com/packages/tools/integration/oculus-integration-82022

2. Enable Integration support in Unity by navigating to Windows -> VRIF settings and clicking on "Oculus Integration" checkbox.

3. Extract the included demo scenes and player prefabs by double clicking the included OculusIntegration.unityPackage

*If you are using HandTracking make sure your build target is set to Android and that "Controllers and Hands" is selected in OVRManager.
** If you are appearing closer to the floor, make sure "Tracking Origin Type" is set to "Floor Level" on OVRManager.

More info can be found at the official docs : http://beardedninjagames.com/vr-framework/docs/#Integrations
